IMPORTAR DADOS ATUALIZADOS DO ACCESS
====================================

OPCAO A — Planilha Excel (.xlsx)  [RECOMENDADO]
------------------------------------------------
1. Tenha o arquivo .xlsx do Access (ex: backups dobanco de dados.xlsx)
2. Ligue o MySQL no XAMPP
3. Execute: sincronizar_access.bat
4. Escolha opcao [3] Planilha Excel
5. Cole o caminho completo do arquivo .xlsx
6. Confirme a sincronizacao

OPCAO B — CSV exportado do Access
----------------------------------
1. No Access: tabela Planilha1 → Exportar → CSV
2. Copie para: advocacia\import\planilha_access.csv
3. Execute sincronizar_access.bat → opcao [1]

O script faz backup automatico antes de sincronizar.

O que acontece:
- Cadastros com o mesmo numero (CADASTRO) sao ATUALIZADOS
- Cadastros novos do Access sao INSERIDOS
- Fotos, PDFs e area juridica do sistema novo sao PRESERVADOS

NAO use migrar.bat para atualizar — ele apaga tudo.
