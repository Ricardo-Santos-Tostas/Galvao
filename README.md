# Moura-Galvao

Sistema de gestão de processos jurídicos — **Moura Galvão Advogados** (PHP + MySQL).

## Início rápido

1. Instale o [XAMPP](https://www.apachefriends.org/)
2. Entre na pasta `advocacia`
3. Ligue o **MySQL** no XAMPP
4. Execute `instalar_novo_pc.bat` (primeira vez) ou `iniciar.bat`
5. Acesse **http://localhost:8080**
